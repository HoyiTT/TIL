# set을 이용한 연산

sorted를 활용해 set에서 - 연산을 할 수 있다.

``` python
ori = set(i for i in range(1, 10001))

data = set()

for i in range(1, 10001):
    tmp = i
    i = str(i)
    for s in range(len(i)):
        tmp += int(i[s])
    data.add(tmp)

ans = sorted(ori - data)
```

# combinations 활용

``` python
from itertools import combinations
chickens = [(0,1), (0,2),(0,3)]
M = 2
for chicken_comb in combinations(chickens, M):
    print(chicken_comb)
```
실행 시

```
((0, 1), (0, 2))
((0, 1), (0, 3))
((0, 2), (0, 3))
```

와 같이 조합을 구할 수 있다.
