# Welcome Page

resoureces/static안에 index.html 파일을 생성해 넣어주기만 하면 된다.

resoureces/static/index.html
```
<!DOCTYPE HTML>
<html>
<head>
    <title>Hello</title>
    <meta http-equiv="Content-Type" content="text/html; charset=UTF-8" /> </head>
<body>
Hello World!!
</body>
</html>

```
