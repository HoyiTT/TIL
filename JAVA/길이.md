# 길이

## length

-arrays(배열)의 길이를 반환한다.

```java 
int[] arr = {1,2,3,4,5};
int length = arr.length;
```

## length()

-String의 길이를 반환한다.

```java
String str = "hello";
int length = str.length();
```

## size()

-Collection의 길이를 반환한다.

```java 
List<String> list = new ArrayList<>();
list.add("hello");
list.add("world");
int size = list.size();
```
